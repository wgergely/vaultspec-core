# .rules Framework

This project uses the `.rules/` framework for AI-assisted development.

- All significant work follows the Research -> ADR -> Plan -> Execute pipeline.
- Skills, agents, rules, and system prompts are defined in `.rules/` and synced to tool-specific directories.
- Documentation artifacts are persisted in `.docs/` (vault).
- See `.rules/README.md` for the full workflow and agent reference.
